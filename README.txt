# Subnet Calculator

## Overview

The **Subnet Calculator** is a Python-based tool that calculates:
- The **network address**.
- The **broadcast address**.
- The **range of valid host addresses** within a subnet.
- The total **number of usable hosts**.

This project is designed to simplify subnetting tasks, a critical skill in network design and troubleshooting.

---

## Features

- **Network Address**: Calculates the network address based on the given IP and subnet mask.
- **Broadcast Address**: Determines the broadcast address for the subnet.
- **Host Range**: Displays the range of valid host addresses.
- **Host Count**: Computes the total number of usable host addresses.
